# Dataset-Cancer
O conteúdo consiste em análise exploratória de dados utilizando um banco de dados contendo informações clínicas, demográficas e socioeconômicas de pacientes diagnosticados com câncer colorretal. Inicialmente, o conjunto de dados foi inspecionado quanto à sua estrutura, tipos de variáveis, valores faltantes e possíveis inconsistências. Em seguida, foram analisadas variáveis numéricas, como idade, tamanho do tumor, custos de saúde e taxas de incidência e mortalidade, por meio de estatísticas descritivas e visualizações gráficas, permitindo compreender suas distribuições e identificar padrões e possíveis outliers. As variáveis categóricas foram avaliadas por meio da contagem de frequências, possibilitando a caracterização do perfil dos pacientes, dos estágios da doença, dos tipos de tratamento e das condições de acesso à saúde. Também foi analisada a variável alvo relacionada à sobrevivência, a fim de compreender o balanceamento das classes e as chances de sobrevivência dos pacientes. Essa análise forneceu uma visão geral do comportamento dos dados e serviu como base para etapas posteriores de tratamento e modelagem.
